# Prerequisites

First of all you need to create an API KEY at [The API of Movie Database](https://developers.themoviedb.org/3/getting-started/introduction)

Next you need to create an env.local file at the root of your application and put:

REACT_APP_MOVIE_API_KEY=YourApiKey

# Installation

You have to clone the Repository and run

npm install
npm run dev

# Deployment

To deploy the application like if that was a production environment just type:

npm run build
